### technine Smart Access API Server

#### Initialization

> $ npm run init

#### Development

Start Server:

> $ npm run dev

#### AMS

Building assets:

> $ npm run admin:vendor

> $ npm run admin:app

Watch client source code changes and compile:

> $ npm run admin:watch

#### Initialise Database

> import `sql/schema.sql` into database
